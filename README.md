### 1. Run Nodejs server

```bash
cd backend
yarn  # install dependencies
yarn start # run server
```

### 2. Run React client

```bash
cd frontend
yarn  # install dependencies
yarn start # run react app
```

You can use `Keystored.txt` key stored file to access sample wallet
